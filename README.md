## 실행 방법

### 프로젝트 받기 & 환경 설정

`git clone`으로 저장소를 받아옵니다.

```
git clone https://github.com/solidcellaMoon/MGMbot-HackaLearnXKorea2021.git
```

저장소가 저장된 폴더로 이동합니다.

```
cd MGMbot-HackaLearnXKorea2021
```

실행, 빌드를 위해 `npm`을 설치합니다.

```
npm install
```

### 로컬에서 실행하기

```
npm run serve
```
